# Network Analysis of ESANN Conferences

**Authors:** Arya Nair, Jens Christian Claussen  
**Institution:** University of Birmingham, School of Computer Science  

This repository contains the research paper, datasets, and scripts used for analyzing the ESANN conference series (2010–2025). The study covers:

- Topic modeling and temporal evolution of conference themes
- Co-authorship network analysis
- Centrality measures and community detection
- Visualization of topic and collaboration trends

## Files
- `paper.pdf` – Full research paper
- `data/` – Processed datasets
- `code/` – Python scripts for analysis
- `figures/` – Visualizations included in the paper
- `embeddings/` – .npy file
